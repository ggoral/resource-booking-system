# Sinatra App Template

This is a [Sinatra](https://github.com/sinatra/sinatra) application template.

## Installation

Clone this repository.

```bash
$ git clone git@github.com/TTPS-ruby/sinatra_app_template.git
```

Remove the origin:

```bash
$ git remote rm origin
```

And then execute:

```bash
$ bundle
```

## Run the app

```bash
$ bundle exec rackup
```

## Run the test suite

```bash
$ bundle exec rake
```
